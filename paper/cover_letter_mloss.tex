\vspace{0.5em}

\noindent \textbf{What \texttt{causalscale} does:} It is a production-grade Python package for scalable
causal discovery. It provides three composable engines---LowRankGNN (up to $d=100$M), ClusterAware
gating, and MultiScale decomposition---under a unified \texttt{pip install} interface. Unlike existing
tools that collapse under the $O(d^3)$ matrix exponential at $d>150$, causalscale maintains $F_1>0.98$
across all dimensions through low-rank factorization ($W=UV^\top$).

\vspace{0.5em}

\noindent \textbf{Community evidence:} Since its release on HuggingFace and PyPI, \texttt{causalscale}
has received over 50 GitHub stars and has been adopted by computational biology and finance research
groups. The package includes pre-trained causal backbones for DepMap (19,215 genes) and TCGA pan-cancer.

\vspace{0.5em}

\noindent \textbf{Demonstration:} A single \texttt{causalscale} command run across 33 TCGA cancer
transcriptomes ($d=100$, $\sim$5 minutes on a consumer GPU) recovers tissue-specific directionality
of the ARID1A--MTOR causal axis in 21 of 33 cancers---findings validated against independent DepMap
CRISPR dependency data.

\vspace{0.5em}

\noindent \textbf{Reproducibility:} All results can be reproduced via:
\begin{verbatim}
pip install causalscale
python -m causalscale.examples.tcga_pancancer
python -m causalscale.examples.benchmark
\end{verbatim}
